### config file in /asstes
# Config

## WSL_VERSION
Site version.

## WSL_SITENAME
Site name/title

## WSL_PATH_/
Path and filename to CSV files
- RANKINGS
- GAMES
- TCORPS

## WSL_MENU
Menu settings
- u : html file
- n : link name
- i : image for index
- a : alt for image
- c : content for index

```
	[ 'u'=>'index.html','n'=>'Home', 'i'=>'', 'a'=>'', 'c'=>'' ],
    ...
	[ 'u'=>'full-rankings.html','n'=>'Full Rankings', 'i'=>'leaderboard-star.svg', 'a'=>'leaderboard', 'c'=>'Rankings of all corporations.' ],
    ...
```
